# 📊 Visualização de Dados — IMDB

Projeto de visualização interativa de dados do IMDB, desenvolvido como atividade prática da disciplina de Visualização de Dados. Com base no [Banco de Dados do IMDb Filmes](https://basedosdados.org/dataset/6ba4745d-f131-4f8e-9e55-e8416199a6af?table=79de8c5e-9c21-4398-a9fb-bc40e6d6e77f), disponibilizado pela [Base dos Dados](https://basedosdados.org)

## 🔗 Demo ao vivo

👉 **[Acesse a apresentação interativa aqui](https://gustavolg95.github.io/Atividade_Visualizacao_de_Dados/)**

## 📁 Estrutura do Repositório

```
├── index.html    # Apresentação interativa com os gráficos
├── codigo_python_imdb.ipynb  # Notebook Python com a análise e geração dos gráficos
└── README.md
```

## 🛠️ Tecnologias Utilizadas

- **Python** — análise e tratamento dos dados
- **Pandas** — manipulação de dados
- **Plotly** — geração de gráficos interativos
- **Jupyter Notebook** — ambiente de desenvolvimento
- **HTML** — apresentação dos resultados no navegador

## 📈 Sobre o Projeto

A análise explora o dataset do **IMDB** (Internet Movie Database), gerando visualizações interativas que permitem investigar padrões em filmes, séries e produções audiovisuais.

Os gráficos foram exportados do notebook Python para um arquivo HTML autossuficiente, possibilitando a visualização diretamente no navegador sem dependências externas.

## ▶️ Como Executar Localmente

**Para visualizar a apresentação:**

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

**Para reproduzir a análise:**

1. Clone o repositório:
   ```bash
   git clone https://github.com/Gustavolg95/Atividade_Visualizacao_de_Dados.git
   ```

2. Instale as dependências:
   ```bash
   pip install pandas plotly jupyter
   ```

3. Abra o notebook:
   ```bash
   jupyter notebook codigo_python_imdb.ipynb
   ```
